# 🚀 Go Concurrency Patterns

A collection of **basic concurrency patterns in Go**, implemented with simple and practical examples.

This project focuses on understanding how to use **goroutines, channels, and context** to build efficient and safe concurrent programs.

---

## 📂 Examples

- [Confinement](./confinement/README.md) – Safe data sharing between goroutines  
- [Context Good Practice](./context-good-practice/README.md) – Cancellation and timeout handling  
- [Fan-In / Fan-Out](./fan-in-fan-out/README.md) – Parallel work distribution and result merging  
- [For-Select](./for-select/README.md) – Handling multiple channels efficiently  
- [Goroutine Leak](./go-routine-leak/README.md) – Common mistakes and how to prevent leaks  
- [Pipeline](./pipeline/main.go) – Structuring concurrent workflows in stages  

---

## 📌 Note

This repository is meant for **learning and practice**. Each folder contains a focused example of a specific concurrency pattern.
